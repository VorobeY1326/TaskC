Задачка по setcontext/swapcontext/etccontext.

Написана на C, компилировалась с помощью gcc: gcc task.c -o task
Запускалась под Lubuntu и Alpine.
Суть: есть несколько потоков с разными приоритетами, есть система переключения между ними в соответствии с приоритетами.

В начале исходника есть пара переменных, перед которыми комментарий: Variables for tuning.
Они необходимы для того, чтобы настроить задержку внутри потока и задержку таймера, чтобы пронаблюдать переключение потоков.
Текущие значения хорошо работают на Lubuntu, но, на другой системе и другой машине их, возможно, надо менять.
